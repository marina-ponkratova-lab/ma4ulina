# Book Tracker

**Автор:** Мачулина Валентина
**Курс:** Итоговая аттестация по GUI-разработке

## Описание приложения

Графическое приложение для учёта прочитанных книг с возможностью добавления новых книг в список и фильтрацией по жанру и количеству страниц.

## Требования к запуску

- Python 3.8+
- Установленные зависимости (см. requirements.txt)
